(Если был установлен пакет через npm глобально, то сначала удалить его с помощью команды
npm uninstall -g create-react-app)

npx create-react-app my-app  -- устанавливаем приложение
cd my-app  -- переходим в папку приложения
npm start  --  стартуем проект

=====** Компоненты  **=====

- ./src - папк в которой хранятся исходные(рабочие) файлы
--- index.js - самый главный файл приложения в который мы будем собирать все модули(js+css+app.js)
--- app.js - содержит главную разметку нашего приложения
--- app.css - содержит базовые стили нашего приложения
--- setupTests.js и App.test.js- тесты содержит
--- reportWebVitals.js - файл для дополнительных функций
--- logo.svg - логотип приложения

- ./public - содержит статичные файлы, которые не будут меняться(например картинки)
--- index.html - базовый html файл, который нужен, чтобы сайт открыть в браузере

=====** Команды **=====

- npm start - открыть проект и отслеживать файлы
- npm build - сборка проекта
- npm test - тестирование
- npm eject - удаляет все настроенные зависимости, на случай, если я захочу перенастроить это приложение
